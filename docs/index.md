# Orthodox Calendar

A multi-language Orthodox Church calendar app for iOS with daily saints, scripture readings, fasting guidance, and saint biographies.

## Features

- **4 Locales**: Serbian, Russian, English (Old Calendar), English (New Calendar / OCA)
- **Calendar Views**: Monthly grid and daily list views
- **Saints & Feasts**: Complete liturgical calendar with feast classifications
- **Scripture Readings**: Daily Epistle, Gospel, and Old Testament readings with full text
- **Fasting Guide**: 7-level fasting system based on Typikon rules
- **Saint Biographies**: Lives of saints from the Ohrid Prologue and other sources
- **Feast Descriptions**: Explanations for all major fixed and moveable feasts
- **Calendar Reminders**: Add feast days to your iOS calendar
- **Share**: Share daily feast information with others
- **Fully Offline**: All data bundled — no internet required
- **Years 2024-2030**: Pre-computed calendar data for 7 years

## Data Sources

This app uses data from the following publicly available sources:

- **crkvenikalendar.com** — Serbian saint biographies (Ohrid Prologue)
- **pravoslavno.rs** — Serbian saints and scripture readings
- **azbyka.ru** — Russian saint biographies
- **orthocal.info** — English saint biographies and OCA calendar data
- **holytrinityorthodox.com** — English scripture readings

### Algorithms

- **Paschalion**: Meeus algorithm for computing the date of Pascha
- **Typikon Lectionary**: Algorithm for daily readings according to the Typikon
- **Fasting Engine**: 7-level fasting computation based on Typikon rules with SPC/RPC variants

### Content

- **Ohrid Prologue** by St. Nikolai Velimirovich — saint biographies, homilies, and hymns
- **Holy Scripture** — Bible texts in Serbian, Russian, and English
